POS Session and Notes Management

This module Manage POS Session according users and added Notes in KOT receipt and bill receipt.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

To access the features of this module you need to configure it from the backend. Here are some easy steps with which you can configure.

  • Cashier and waiter session is managed from Setting --> Users. Here, Demo user is given rights of cashiers.

Step-By-Step Guidelines

User wise session is managed with the help of this module. There is a field named 'Users' in the pos session. Here, in this example we have choosed 'Demo User' for 'POS session1'. So, 'POS session1' will only be accessed by this user.

User access to their session. 'POS session1' will be accessible when logged in via Demo User.

Cashier and waiter session is managed from Setting --> Users. Here, Demo user is given rights of cashiers.

POS screen displays when note is added while adding the product.

This notes is added in the POS receipt and will be printed once the Order is printed.

When order is created from the POS the notes is added in the orderline.
